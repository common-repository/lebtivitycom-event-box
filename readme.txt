== Description ==

The Event Box is a plugin that enables you to display all of Lebtivity's events on your website. 

If you are a blogger, this plugin will act as a miniature calendar for your loyal visitors. 

Simply customize the widget in the admin page and you're set!


== Installation ==

1. In your wp-admin page, got to: 'Plugins'>'Add New'> Search for "Lebtivity" and install the widget
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In 'Appearance' go to 'Widgets' and drag 'Lebtivity Event Box' to the desired location
1. Customize the Widget
1. Enjoy!
